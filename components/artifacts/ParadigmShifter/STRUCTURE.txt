Dependencies:
=============
- React 19
- Framer Motion (animations)
- Tailwind CSS (styling)
- TerminalWindow from @/components/ui/Terminal

State Management:
=================
ParadigmShifter:
  - mode: 'legacy' | 'vibe' (useState)
  
LegacyState & VibeState:
  - No local state (pure presentation)
  - Receive animations from parent variants

Animation Flow:
===============
1. User clicks toggle button
2. mode state updates (legacy ↔ vibe)
3. AnimatePresence detects key change
4. Exit animation plays on current component (200ms fade)
5. Entry animation plays on new component:
   - Container: stagger children (0.1s delay)
   - Badge: 400ms fade-up
   - Title: 400ms fade-up (staggered)
   - Bullets: 400ms fade-up (staggered)
   - Terminal: Lines animate in sequence (100ms delay each)

Responsive Breakpoints:
=======================
